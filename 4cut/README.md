# Dockerfile 빌드하고 실행하는 방법

일단 도커 다운하세요 ㄷㄷ

**빌드하기**
```ssh
docker-compose build
```

**실행하기**
```ssh
docker-compose up
```

**종료하기**
```ssh
docker-compose down
```

-> 기본적으로 ctrl + c 종료 가능함
